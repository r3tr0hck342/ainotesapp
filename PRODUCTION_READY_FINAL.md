## 🚀 Build Commands

### Install Dependencies
```bash
npm install
```

### Build React App
```bash
npm run build
```

### Build for Linux
```bash
npm run build:linux
```
Creates:
- `dist-electron/AI-Notes-1.0.0.AppImage`
- `dist-electron/ai-notes-app_1.0.0_amd64.deb`

### Build for macOS
```bash
npm run build:mac
```
Creates:
- `dist-electron/AI Notes.dmg`
- `dist-electron/AI Notes.app.zip`

### Build for Both Platforms
```bash
npm run build:all
```
